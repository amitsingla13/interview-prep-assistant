Privacy Policy

Last updated: June 2025

1. Introduction

Interview Preparation Assistant ("we", "our", "the app") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our application.

2. Information We Collect

We collect the following types of information:

  • Voice Data: When you use the voice feature, your audio is recorded and sent to OpenAI's Whisper API for transcription. Audio is processed in real-time and is NOT stored permanently on our servers.
  • Text Messages: Messages you type are sent to OpenAI's GPT API for generating responses. These are stored temporarily in server memory during your session and deleted when you disconnect.
  • Session Data: We store temporary session information (conversation history, mode selection, timestamps) in server memory. This data is automatically deleted when your session ends or expires (1 hour).

3. How We Use Your Information

  • To provide AI-powered interview practice, language practice, and general chat features
  • To transcribe your voice input into text (via OpenAI Whisper)
  • To generate AI responses (via OpenAI GPT)
  • To generate text-to-speech audio (via OpenAI TTS)
  • To moderate content for safety (via OpenAI Moderation API)

4. Data Storage & Retention

  • All conversation data is stored only in server memory and is never written to a database or file system.
  • Session data is automatically deleted after 1 hour of inactivity or when you close the app.
  • TTS audio responses may be cached temporarily in server memory to reduce costs. Cache is purged when the server restarts.
  • We do NOT store any user accounts, passwords, email addresses, or personal identifiers.

5. Third-Party Services

We use the following third-party services:

  • OpenAI API: For speech-to-text (Whisper), text generation (GPT), text-to-speech (TTS), and content moderation. OpenAI's data usage policies apply. See OpenAI's Privacy Policy.
  • Render.com: For hosting. See Render's Privacy Policy.
  • Cloudflare CDN: For serving static assets (Font Awesome icons, Socket.IO library).

6. Data Security

We implement several security measures:

  • HTTPS encryption for all data in transit
  • Rate limiting to prevent abuse
  • Content moderation to filter harmful content
  • Input validation and sanitization
  • No permanent data storage — everything is in-memory and ephemeral

7. Children's Privacy

This app is not directed at children under 13. We do not knowingly collect information from children under 13. If you are a parent and believe your child has provided us information, please contact us.

8. Microphone Permission

The app requests microphone access only when you choose to use the voice feature. Microphone access is optional — you can use the app entirely via text input. Audio recording only occurs while the microphone button is active.

9. Your Rights

  • You can reset your session at any time using the "Reset" button, which immediately deletes all your conversation data.
  • Since we don't store any persistent personal data, there is nothing to delete after your session ends.
  • You can use the app without providing any personal information.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected on this page with an updated date.
